# MaClaw（码卡龙）使用说明书

[❓ 常见问题 (FAQ)](faq.md)

欢迎使用 **MaClaw（码卡龙）** —— 你的**编程小龙虾**。无论是专业软件开发者还是普通白领，它都是你身边最贴心的 AI 搭档。它不仅懂代码，更懂你：可定制人格、长期记忆你的偏好，在你工作时给予精神陪伴，在你忙碌时自动接管任务。集三态编程、智能体编排、人格系统、记忆管理、SSH/浏览器监管于一体，**软件开发、白领办公、日常事务**统统拿下。

以下是详细的操作指南：

## 1. 首次启动与 Onboarding 四步走

首次运行 MaClaw 时，会自动弹出引导向导，只需四步即可完成所有配置：

### 第一步：注册账号
*   输入邮箱地址完成注册（部分地区需要邀请码）
*   注册成功后自动启用远程编程能力

### 第二步：选择界面模式
*   **专业模式 (Pro)**：面向开发者，完整暴露所有高级功能（三态编程、Swarm 编排、SSH/浏览器监管、MCP 管理、记忆系统等）
*   **简洁模式 (Simple)**：面向白领与普通用户，精简界面零门槛，聚焦对话式 AI 交互，写文档、做报表、查资料样样行
*   可在设置中随时一键切换——**开发者用专业模式写代码，白领用简洁模式办公，一家人整整齐齐。**

### 第三步：配置 AI 模型
*   从预设 LLM 服务商中选择（如 GLM、DeepSeek 等）
*   输入 API Key 并测试连通性
*   也支持 OAuth 一键登录和免费体验通道

### 第四步：绑定微信（可选）
*   扫描二维码即可将小龙虾绑定到你的微信
*   绑定后可随时随地通过微信与小龙虾对话
*   手机端发消息就能指挥小龙虾帮你写代码、查资料、处理文档

> 四步完成，电脑上用界面操作，手机上用微信指挥，随时随地都在线。

## 2. 启动与环境检测

启动 MaClaw 时，会自动检查您的系统环境：
*   **依赖检测**：检测 Node.js 及其他必要运行时。
*   **工具安装**：自动检测并安装/更新 `claude-code`、`codex`、`gemini-cli`、`opencode`、`codebuddy`、`qodercli` 等工具到最新版本。
*   **启动窗口**：显示进度窗口，展示环境准备状态。

## 2. 三态编程模式

MaClaw 提供三种互补的编程模式，满足不同开发场景：

### 2.1 本地 AI 编程
AI CLI 工具在本机直接运行，完整访问文件系统与开发环境。

1.  在侧边栏选择目标 AI 工具（Claude、Codex、Gemini 等）。
2.  在 "Vibe Coding" 区域选择项目。
3.  点击 **"Launch"**，弹出预配置终端窗口并自动运行。

**适用场景**：日常编码、调试、重构、代码审查。

### 2.2 远程 AI 编程
通过 MaClaw Hub 提供浏览器远程访问，支持 PWA 及移动端。

1.  确保 MaClaw Hub 已部署并可访问。
2.  在浏览器中打开 Hub 地址，使用邮箱和邀请码激活。
3.  选择会话并连接，实现实时远程编程。

**适用场景**：远程办公、跨设备协作、移动端紧急处理。

### 2.3 AI 自动编排编程
Swarm 编排器智能拆分任务，多智能体并行执行并自动合并结果。

1.  在编排面板中选择运作模式：
    *   **Greenfield**：从零开始构建项目
    *   **Maintenance**：对现有代码库进行增量维护
2.  输入需求描述，编排器自动拆分为子任务。
3.  最多 5 个 AI 开发者智能体并发执行，含冲突检测与自动合并。
4.  每任务最多 5 轮反馈迭代，持续优化输出。

**适用场景**：大型需求开发、批量重构、自动化流水线。

## 3. 智能体系统

### 3.1 人格配置 —— 养成你的编程小龙虾
自定义小龙虾的性格与身份，让它成为专属于你的编程伙伴：

1.  打开 **MaclawRolePanel**（人格配置面板）。
2.  设置以下参数：
    *   **角色名称**：自定义小龙虾的名字（默认："MaClaw"）
    *   **角色描述**：详细的角色定义（默认："一个尽心尽责无所不能的软件开发管家"）
3.  角色信息将自动注入系统提示词，影响所有交互行为，让小龙虾始终保持一致的性格。
4.  支持在对话中动态临时覆盖角色设定，适配不同场景（严肃代码审查 / 轻松结对编程）。
5.  在漫长的编码夜中，小龙虾始终在线——随时响应、永远耐心、始终陪伴。

### 3.2 后台任务管理
系统通过槽位机制管理并发任务：

| 槽位类型 | 最大并发数 | 说明 |
|----------|-----------|------|
| 编程任务 (Coding) | 1 | 交互式编程会话 |
| 定时任务 (Scheduled) | 1 | 按计划自动执行 |
| 自动任务 (Auto) | 1 | ClawNet 自动拾取任务 |
| SSH 会话 | 10 | 远程终端连接 |
| 浏览器任务 | 2 | 浏览器自动化 |

后台任务支持实时进度监控，长时间运行的任务可随时查看状态。

## 4. 记忆管理系统 —— 小龙虾记得你的一切

MaClaw 提供长期记忆能力，让小龙虾跨会话记住你的偏好和项目知识，越用越懂你：

### 4.1 记忆分类
系统维护六类结构化记忆：

| 类别 | 说明 | 权重 |
|------|------|------|
| 自我认知 | 智能体身份与性格定义 | 4.0 |
| 用户事实 | 关于用户的基本信息 | 2.0 |
| 用户指令 | 用户明确下达的规则 | 3.0 |
| 偏好设定 | 编码风格、工具偏好等 | 2.0 |
| 项目知识 | 项目特定的技术决策和架构信息 | 2.0 |
| 会话检查点 | 跨会话的上下文延续 | 2.0 |

### 4.2 记忆操作
*   **置顶 (Pin)**：将重要记忆标记为不可淘汰
*   **归档 (Archive)**：将不常用记忆转入冷存储
*   **压缩 (Compress)**：LLM 驱动的去重 + 语义合并 + 智能压缩
*   **垃圾回收 (GC)**：基于 LRU 策略自动淘汰过期记忆（自我认知类记忆永不淘汰）

### 4.3 记忆检索
采用三维度评分模型：`Score = w1 × 时效性 + w2 × 重要性 + w3 × 相关性`

*   **时效性**：基于更新时间的指数衰减
*   **重要性**：类别权重 + 访问次数对数
*   **相关性**：BM25 关键词匹配 + 向量语义相似度 + 项目亲和度

## 5. SSH 运行监管（差异化能力）

MaClaw 独创的 SSH 运行监管系统，为软件运维提供端到端的远程管控能力，这是区别于同类产品的核心差异化特性：

### 5.1 远程会话管理
1.  在 SSH 面板中添加远程主机信息（地址、端口、认证方式）。
2.  建立连接后，会话纳入后台任务管理（最多 10 个并发）。
3.  实时监控远程操作状态与进程信息。

### 5.2 运维自动化
*   结合智能体实现远程部署、日志分析、故障诊断
*   支持批量命令执行与结果收集
*   操作审计日志全程记录

## 6. 浏览器运行监管（差异化能力）

MaClaw 独创的浏览器运行监管系统，为自动化测试与业务流程自动化提供核心支撑，这是区别于同类产品的核心差异化特性：

### 6.1 流程录制
1.  打开浏览器录制面板，点击 **"开始录制"**。
2.  在浏览器中执行操作，系统自动记录每一步交互。
3.  点击 **"停止录制"** 保存为可回放的流程文件。

### 6.2 流程回放
1.  选择已录制的流程文件。
2.  可设置 **变量覆盖** 进行参数化回放。
3.  回放任务以后台任务方式运行，支持暂停/恢复/取消。
4.  支持定时触发，按计划自动执行。

### 6.3 OCR 与验证
*   支持 RapidOCR 及 LLM 视觉模型进行屏幕文字提取
*   内置任务验证机制，确认自动化操作执行正确
*   失败自动重试策略

## 7. 侧边栏导航与工具管理

### 7.1 工具导航
侧边栏提供快速切换：
*   **Claude**：配置和启动 Anthropic Claude Code
*   **Codex**：配置和启动 OpenAI Codex CLI
*   **Gemini**：配置和启动 Google Gemini CLI
*   **OpenCode**：配置和启动 OpenCode AI 辅助工具
*   **CodeBuddy**：配置和启动 CodeBuddy 编程助手
*   **Qoder**：配置和启动 Qoder CLI 编程助手
*   **技能 (Skills)**：管理 AI 编程助手的扩展能力

### 7.2 技能管理
1.  **添加技能**：支持 **Skill ID (Address)** 和 **Zip 包** 两种格式。
2.  **技能共享**：Zip 技能存储在全局仓库，所有工具共享。
3.  **兼容性检查**：自动识别工具特性，不兼容时给出提示。
4.  **系统技能**：内置官方文档等默认技能，开箱即用。

## 8. 模型配置

1.  在侧边栏选择 AI 工具，找到 **"模型设置"** 区域。
2.  **服务商选择**：支持 GLM、Kimi、Doubao、MiniMax、DeepSeek 等预设服务商。
3.  **"原厂" (Original) 模式**：使用官方默认配置，自动清除自定义代理与环境变量。
4.  **API Key**：在输入框中粘贴密钥，配置后自动保存。
5.  **智能同步**：同一服务商的 Key 在不同工具间自动同步。

## 9. 多项目管理

### 9.1 切换项目
在 **"Vibe Coding"** 区域查看项目选项卡，点击切换。

### 9.2 项目管理
点击 **"管理项目"** 按钮可添加、重命名或删除项目。

### 9.3 项目参数
1.  **项目目录**：点击 "更改" 选择代码文件夹。
2.  **启动参数**：
    *   **Yolo 模式**：跳过所有权限询问（谨慎使用）
    *   **管理员权限 (Windows)**：以管理员身份启动终端
3.  **Python 环境**：自动检测 Conda/Anaconda 环境，支持独立 Python 运行环境。

## 10. 工具路由与 MCP 集成

### 10.1 智能工具路由
*   系统通过 BM25 + 向量语义双重检索自动匹配最相关工具
*   每请求最多 30 个工具，按相关性动态分配预算
*   详细的工具路由日志记录于 `~/.maclaw/logs/tool_route.log`

### 10.2 MCP 服务器管理
*   **本地 MCP 服务器**：Stdio 协议接入，支持自动发现与健康监控
*   **远程 MCP 服务器**：HTTP 协议接入，支持认证与 60 秒周期健康检查
*   工具定义自动提取与注册

## 11. 其他功能

*   **状态栏**：界面底部显示实时操作反馈和错误信息
*   **多语言切换**：在标题栏或设置中切换界面语言
*   **检查更新**：获取 MaClaw 最新版本
*   **系统托盘**：右键托盘图标可快速启动工具或退出程序
*   **安全框架**：风险评估引擎、审计日志、安全防火墙与细粒度权限控制
